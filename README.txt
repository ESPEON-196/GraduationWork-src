各ファイル説明
・メインファイル
　- main.py:このファイルを実行，指定された仮想環境における，最大遅延時間を削減するエッジサーバの配置場所及びジョブの割り当て先を決定し，値を評価する．ジョブ割り当て手法とエッジサーバ配置調整手法を反復し、シミュレーテッドアニーリングを用いて配置と割り当ての組み合わせの解を探索する．
　- allocation.py:ジョブ割り当て手法．
　- replacement.py:エッジサーバ配置調整手法．
・関数ファイル(my_module内)
　- topologymaking.py:指定されたジョブの到着率の平均，距離の平均を元に，ノード数Nのランダムなネットワークトポロジを作成．
　- calculator.py:主に待ち行列理論に関する複雑な計算を行う．
